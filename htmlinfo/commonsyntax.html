Common Syntax for HTML

HTML Boilerplate

The HTML Boilerplate is what must be entered before creating anything in HTML. The quickhand key to it is ! on the first line of an HTML document. The boiler plate itself consists of an html, head, and body element tag.

Divisions

Element tags such as div (block display) and span (inline display) are used to divide up different sections of content. Tags such as p and hX cannot be used to divide up sections. A common use for dividing element tags is to create flex boxes
